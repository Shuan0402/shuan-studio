# 經歷：自動化測試實習生 @ 威聯通科技 (QNAP)
## 基本資訊
- 時間：2025.01 - 2025.06 
- 部門：研發品管中心 / 設計驗證二部 / 驗證二課 
- 類別：工作經驗

## 詳細職責與技術貢獻
實習期間共處理 103 項工作單，涵蓋開發、環境建置、測試驗證與 Bug 修復 。

1. 資安流程自動化 (APP LAB 導入 Trivy)
    - 開發動機：補足 APP LAB 流程中缺少的資安檢測能力，主動偵測漏洞與敏感資訊 。
    - 實作細節：
        - 整合 Trivy 掃描工具至 Jenkins 工作流，實現兩次掃描流程（Pretest 與 Test）。
        - 開發自動化腳本分析 Jenkins Console Output，讀取漏洞總數 (Total Vulnerabilities) 與敏感資訊總數 (Total Secrets) 。
        - 設計 API 規格進行數據回傳與對比，產出資安報告並判斷測試是否 PASS 。
2. 軟體更新自動化 (QVR Pro Client 批次更新)
    - 開發動機：將原本需要工程師手動更新多台機器的流程轉向自動化，支援機器並行處理以節省時間並減少失誤 。
    - 實作細節：
        - 使用 Python 與 Jenkins 腳本實現自動化下載、安裝指定版本、版本驗證與清理流程 。
        - 加入「機器狀態檢查機制」：判斷機器是否在線或閒置，若在使用中則暫存任務稍後執行 。
        - 成功達成 100% 安裝成功率與自動化報告彙整 。
3. 跨產品自動化測試開發
    - 支援產品：QVR Pro Client (Windows/iOS/Android)、QVR Human、Qumagie 。
    - 技術應用：
        - 使用 Airtest 與 Xpath 定位進行 UI 自動化測試 。
        - 撰寫 API 驗證腳本執行功能測試 。
        - 執行手動驗證與 Bug 分析，透過 Jira 追蹤錯誤修復進度 。
        
## 學習與成長
- 技術深化：掌握了 CI/CD 實務流程，能獨立撰寫 Jenkins Pipeline 與自動化測試腳本 。
- 問題解決：學會透過手動驗證與 Debug 交叉比對問題，並修正跨平台資料傳輸與權限設定（如 NAS 自動連線） 。
- AI 應用：在開發過程中熟練運用 AI 輔助工具，大幅提升程式碼編寫與錯誤排查的效率 。
